# Student Grade Tracker

## Project Overview
Student Grade Tracker is a simple Java console-based application used to manage student grades.  
The program allows users to enter student names and marks, then calculates:

- Average score
- Highest score
- Lowest score

It also displays a summary report of all students.

---

# Features

- Add multiple student records
- Store data using ArrayList
- Calculate average marks
- Find highest and lowest scores
- Display summary report
- Console-based interface
- Simple and beginner-friendly Java project
